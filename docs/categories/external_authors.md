# External Authors

Vendored third-party utilities from MATLAB File Exchange and community sources.

```{note}
These files are **not original lab work.** They're included here to keep this
toolbox self-contained — some are widely-used, others are small utilities
that just aren't worth reimplementing. Each file retains its original author
attribution in its header.
```

## Contents

| Function | Origin / purpose |
|---|---|
| {mat:func}`findpeaks_extents` | Peak-finding with peak-extent metadata |
| {mat:func}`invprctile` | Inverse of `prctile` — returns the percentile rank of a value |
| {mat:func}`precise_datatip` | Precise data-tip display on plots |
| {mat:func}`progressbar` | Text/GUI progress bar |
| {mat:func}`real2rgb` | Map real-valued matrix to RGB via colormap |
| {mat:func}`sigstar` | Add significance stars between bar groups |
| {mat:func}`splitvec` | Split a vector at discontinuities |
| {mat:func}`suptitle` | Super-title across a figure with subplots |
| {mat:func}`write_edf` | Write a European Data Format (EDF) file |
| {mat:func}`xml2struct` | Parse XML into a MATLAB struct |
